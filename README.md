# Urban Mobility Shock Absorber

Urban transportation systems are vulnerable to sudden disruptions. When transit services fail, commuters must rapidly adapt using alternative mobility options. This project investigates how London's bike-sharing network responds to London Underground strike events and measures the extent to which bike-sharing acts as an urban "shock absorber" during transportation disruptions.

## Research Question

To what extent does bike-sharing usage increase during London Underground strike events, and how effectively does the Santander Cycles network absorb displaced transit demand?

## Methodology
Identify strike events.
Establish baseline bike demand.
Measure strike-day demand.
Calculate mobility shifts.
Perform spatial analysis.
Estimate absorption capacity.
